# 👩‍💻 Personal Portfolio Website

This is a simple and responsive personal portfolio website built using HTML and CSS.  
It showcases basic information, skills, and projects in a clean layout.

---

## ✨ Features

- Simple and modern UI design  
- Responsive layout  
- About, Skills & Projects sections  
- Beginner-friendly structure  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  

---

## 🎯 Purpose

This project was created to practice:
- Building real-world portfolio layouts  
- Structuring multi-section webpages  
- Improving frontend design skills  

---


## 🚀 How to Run

1. Clone or download the repository  
2. Open `index.html` in browser  
3. Explore the portfolio website
